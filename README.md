# yt-download

Ferramenta para realizar download de vídeos do YouTube e converterter para .mp3

## Dependências

| Nome    | Versão |
|:-------:|:------:|
| python  | 3.6.9  |
| pip     | 9.0.1  |
| pytube  | 10.4.1 |
| moviepy | 1.0.3  |

Se estiver em ambiente Linux é necessário ter instalado o ffmpeg.
